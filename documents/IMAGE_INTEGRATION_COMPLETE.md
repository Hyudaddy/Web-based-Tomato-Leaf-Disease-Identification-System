## How It Works

The Information page now:

1. **Maps disease names to folder names** using the `diseaseImageMap` object
2. **Stores all image filenames** in the `diseaseImages` object
3. **Retrieves images dynamically** with the `getDiseaseFolderImages()` function
4. **Renders actual images** using Next.js `Image` component with proper paths: `/diseases/{folderName}/{imageName}`
